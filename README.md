# ClinicalLibrary

The goal is to build clinical data held by K-CURE participating medical institutions into an in-hospital clinical library through a standardization process.

1.암임상라이브러리
  - 표준화된 암임상 스키마 정보
  
2.표준사전
  - 데이터표준화 정보
  
3.공통코드
  - 코드테이블 및 공통코드 정보
  
4.중앙암등록
  - 국가암등록 DB 스키마 정보
  
5.메타통계DB
  - 메타통계 DB스키마 정보
  
6.BaseLineDB
  - OMOP-CDM 스키마 정보
